           2) /admin/add     添加一个管理员账号
                 -- 参数       name, login_name, password(md5)

                 -- 参数       id, enable=1/0
                 -- 返回值    success

           4) /admin/remove  删除一个管理员账号
                 -- 参数       id
                 -- 返回值    success

           5) /admin/edit    编辑一个管理员账号
                 -- 参数       id, name, login_name, password(md5)
                 -- 返回值    {id, name, login_name, email, last_login_time, ip }
